🚀 Getting Started

Vercel es una plataforma para desarrolladores que provee las herramientas e infraestructuras necesarias para construir y desplegar aplicaciones web de forma más rápida sin configuraciones adicionales.

  1. Crear una cuenta en Vercel con GitHub, GitLab o Bitbucket.
  2. Importar un repositorio del proyecto directamente desde la plataforma.
  3. Vercel detecta automáticamente el framework y configura el despliegue.
  4. Al finalizar, genera una URL única para visualizar el proyecto en línea.

Cada vez que se haga un push en la rama principal del repositorio, Vercel actualizará automáticamente la aplicación desplegada.
